상, 하위 컴포넌트 관계
vue.js의 컴포넌트는 각각 고유한 scope를 갖고 있기에, 값을 전달하려면 바로 상위(부모)-하위(자식)컴포넌트 간의 데이터 전달 방식을 따라야 함
-자식 컴포넌트 : 등록된 지역, 전역 컴포넌트
-부모 컴포넌트(상위 컴포넌트) : 자식 컴포넌트(하위 컴포넌트)를 등록한 인스턴스

-상위 -> 하위 (props 속성 전달)
-하위 -> 상위 (이벤트만 전달 가능)
*상위에서 하위 컴포넌트로 데이터 전달하기 : props 속성 사용
1. props 속성에 정의
2. 컴포넌트 태그에 v-bind 속성 추가
